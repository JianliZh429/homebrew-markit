# Homebrew Tap for Markit

This is a Homebrew tap for installing [Markit](https://github.com/JianliZh429/markit) - a modern cross-platform Markdown editor.

## Installation

```bash
# Tap this repository
brew tap JianliZh429/markit

# Install Markit
brew install --cask markit
```

## Uninstall

```bash
brew uninstall --cask markit
brew untap JianliZh429/markit
```

## About Markit

Markit is a modern, cross-platform Markdown editor built with Electron and TypeScript.

### Features

- **Dual-mode editing**: Switch between raw markdown editor and live preview
- **Real-time preview**: Instant markdown rendering with syntax highlighting
- **Smart paste**: Automatic HTML-to-Markdown conversion
- **File explorer**: Built-in tree view with folder navigation
- **Search**: Local and global search across files
- **Auto-save**: Configurable automatic saving
- **Export to HTML**: Export styled HTML with print-ready formatting
- **Image Drag-and-Drop**: Auto-save images to `.assets/` folders

For more information, see the [Markit repository](https://github.com/JianliZh429/markit).
